# make-xadmin
自制Django的后台组件
 
xadmin组件主要是模仿django自带的admin后台，拿来练练手。

admin后台最显著的好处根据模型自动生成增删改查界面

这个项目也主要是粗糙地实现这一功能

### xadmin主要代码位于 ./xadmin/service/xadmin.py中（建立app是为了方便测试）

IDE：Pycharm    Python版本: 3.6
